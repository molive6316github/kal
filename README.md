# Kalbot

> A slack bot for kingdoms, elections, trade, war, and more.



![License](https://img.shields.io/badge/license-MIT-blue)



## Commands

```
-/kal-ping - get the bot's ping
-/kal-help - get this list!
-/kal-join - join the game, and get assigned a kingdom!
-/kal-actions - get a list of actions, and click them to do them!
```



<br>

> **Note:**
> Minor amount of ai were used for json structing, planning, and debugging.
